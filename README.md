# Proyecto: Reconocimiento de Voz (Web)

Pequeña aplicación web para demostración de reconocimiento de voz en el navegador.

## Requisitos
- Navegador moderno con soporte para la API de reconocimiento de voz (p. ej. Chrome).
- Node.js / `npx` (opcional, solo para servir los archivos localmente).

## Ejecutar (modo local)
1. En la raíz del proyecto, inicia un servidor estático (ejemplo con `http-server`):

```bash
npx http-server -p 8080
```

2. Abre tu navegador en http://localhost:8080 y prueba la aplicación.

## Estructura de archivos
- `index.html` — Interfaz de usuario.
- `script.js` — Lógica JavaScript (reconocimiento de voz y control UI).
- `style.css` — Estilos CSS.
- `README.md` — Este archivo.

## Uso
- Pulsa el botón de grabar para iniciar el reconocimiento de voz.
- Revisa la consola del navegador para mensajes y errores.
